Data integrity notes

  • FWA timestamp caveat: A later FWA row is not proof of a full re-assessment. Any single field edit resets the submission timestamp, so improvement flags are provisional.
  • Trial status: The "at least one domain improved" measure is exploratory only. The official KPI 2 requires composite FSS improvement (post > pre total).
  • Cohort anchor: This trial anchors the 91-180 day post window to each mom's derived intake date, not pairing start. The production KPI 2 uses pairing start. Cohorts may differ.
  • Strict KPI 2 tag: Moms tagged strict = 1 are in a period-active pairing (Jan 1 - Jun 30 2026), not PS-migrated. Matches production cohort logic with PERIOD_END = 2026-06-30.
  • Sample size: n < 10 = too few to interpret; n < 20 = interpret with caution; n ≥ 20 = reportable. Check the strict subset especially.
